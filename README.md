# react-cli

A simple CLI for scaffolding React.js projects.

## Install
```bash
$ npm install -g @ice-point/react-cli
```

## Usage
```bash
$ react init <project-name>
```

Example:

```bash
$ react init my-project
$ cd my-project
$ npm start
```
